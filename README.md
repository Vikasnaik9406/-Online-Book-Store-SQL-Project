## Online-Book-Store-SQL-Project
The Online Book Store is a relational database project built using SQL. It simulates a real-world book store system where customers can browse books, place orders, and manage transactions.  This project demonstrates database design, relationships, constraints, joins, aggregations, window functions, and real-world query solving.

## 🎯 Objectives

Design normalized relational database

Implement primary and foreign key relationships

Perform CRUD operations

Write complex SQL queries

Analyze sales and customer data

## 📁 Project Structure

- Online Book Store Project.sql → Database schema and queries
- Books.csv → Books dataset
- Customers.csv → Customer dataset
- Orders.csv → Orders dataset
- README.md → Documentation

## 🔗 Database Relationships

One Customer → Many Orders

One Order → Many Books

Books linked using Foreign Keys

Enforced Referential Integrity

## 📊 Business Problems Solved

Performed genre-wise sales aggregation for revenue insights.

Analyzed repeat customer behavior using HAVING clause.

Identified top-performing products using ranking logic.

Evaluated inventory balance using LEFT JOIN and aggregate functions.

Conducted customer spending analysis using grouped aggregations.
